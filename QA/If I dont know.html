If I don’t know where my files on the device are, how do I specify the folder when creating the index?

You can create all indexes via:
Menu ->"Create/Maintain index"->"Create index"
Or
Menu ->"Create/Maintain index"->"Maintain index"->"Create filename index" and "Scan all files to index"

The function of "Scan all files to index" will scan the entire device and list all folders containing any supported files. You can select one or more folders to index.

If you still have no idea of which folders to index, you can select all folders and index them at a time.
