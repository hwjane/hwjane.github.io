When I open a result file with the related app, can DocSearch+ open the file to the keyword position instead of the first page?

There seems to be some misunderstanding. DocSearch  is just a search tool. When it finds result files, you open these files with other related apps(not DocSearch+). The related apps come from other developers. Therefore, DocSearch+ can't control how they open these files. 
For example, if you open a pdf file with "Adobe PDF Reader", only "Adobe PDF Reader" can decide  how to open this file, not DocSearch+.
