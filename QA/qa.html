Q. I have installed DocSearch plus, but I cannot find anything. What are the correct steps to use it?

A. Before starting your first search, you need to create indexes for your files. Please follow  the steps below:
(You can also refer to "Menu->Help->Brief User Guide->Page 2 & 3 for details")

Step 1:
Create the indexes via:
Menu ->"Create/Maintain index"->"Create/Remove index manually"->"Create filename index" and "Create file content index"

You will see the following screen and follow the instructions described in the screenshot.



Step 2:
Use a keyword to search your files on the main screen of this app.

Detailed steps as following:
1. Enter keywords (e.g. "happy")
2. Press the blue "search icon" on the right.
3. All files containing "happy" will show.
4. Click the file name, you can see the file content in the lower half of the screen.
(For more functions on the main screen, please refer to "Menu -> Help -> Brief User Guide -> Page 5")




Q: If I don’t know where my files on the device are, how do I specify the folder when creating the index?

A: You can create indexes via:
Menu ->"Create/Maintain index"->"Create/Remove index manually"->"Create filename index" and "Scan all files to index"

The function of "Scan all files to index" will scan the entire device and list all folders containing any supported files. You can select one or more folders to index.

If you still have no idea of which folders to index, you can select all folders and index them at a time.

Q. How do I create index for entire device?

A: If you want to create indexes for entire storage, you can execute:
 "Create/Maintain index"->"Create/Remove index manually"->"scan all files to index"
It will scan all folders and show all folders containing any supported file (txt, pdf, epub, docx, xlsx, odt, etc.).
And you can create indexes for multiple folders at a time.

Q. What is the difference between file name index and file content index?

A. You can refer to "Menu->Help->Brief User Guide->Page 4 - Two type of indexes"
It explains the difference between filename index and file content index.

Q. Do I need to specify the level of the directory hierarchy to start the search.

A. You don’t need to specify any directory, just start searching on the main screen of this app. This app will search all the indexes you created.

Q. After I find the files I want, how can I use them?

A.
1. You can view the contents of these files in plain text mode, and easily locate the search keywords in the file.
a. In brief text mode, you can view a short paragraph of content where all keywords are located
b. In the full text mode, you can view the entire content of the document, and easily switch the position between each keyword.

2. You can copy, move, delete or share these files

3. You can open these files with other external related apps, for example, open PDF files with a PDF reader

(For more operations on the main screen, please refer to "Menu -> Help -> Brief User Guide -> Page 5")






Q. When I opened a result file with the related app, can DocSearch+ open the file to the keyword position instead of the first page

A. There seems to be some misunderstanding, DocSearch  is just a search tool, When it finds result files, and you open these files with other related apps(not DocSearch+). The related apps come from other developers. Therefore, DocSearch+ can't control how they open these files. 
For example, if you open a pdf file with "Adobe PDF Reader", only "Adobe PDF Reader" can decide  how to open this file, not DocSearch+.
