Privacy Policy

What information do we collect?
We collect information from you only when this Software crash.

The crash information includes:
1. crash description containing the basic information about the crash:
(a) device model and brand
(b) crash time
(c) Software name
(d) country and language
(e) you may be asked to enter your e-mail address and short problem description. this information is also included into the crash description

2. crash minidump file which contains technical information about exception in the Software:
(a) processor information
(b) memory information
(c) operating system version
(d) process and thread information
(e) stack traces for each thread
(f) the list of loaded modules


What do we use your information for?
Any of the information we collect from you may be used in one of the following ways:

1. to improve the Software (we continually strive to improve the Software based on the information and feedback we receive from you)
2. to find bugs and fix them in the Software


Do we disclose any information to outside parties?
We do not sell, trade, or otherwise transfer to outside parties your personally identifiable information.
